# **App Name**: Thrive Daily

## Core Features:

- Daily Worries List: Users can create and manage a list of daily worries to help process and address anxieties.
- Gratitude List: Users can create a daily gratitude list to focus on positive aspects and improve overall well-being.
- Goals List: Users can define and track their goals in a dedicated list, promoting a sense of direction and accomplishment.
- Tasks List: Users can create and manage task lists to organize their daily activities and increase productivity.
- Meditation Section: A meditation section featuring a timer and guided scripts to facilitate relaxation and mindfulness practices. 
- Daily Inspirational Quote: Receive a daily inspirational quote to motivate and uplift the user; a tool may select a quote that the LLM recognizes as relevant to entries made by the user that day.

## Style Guidelines:

- Primary color: Soft lavender (#D1B0FF) to promote calm and relaxation.
- Background color: Very light grayish-lavender (#F7F4FB), almost white.
- Accent color: Muted plum (#946CB2) for subtle emphasis and interactive elements.
- Body and headline font: 'Alegreya', a humanist serif with an elegant, intellectual, contemporary feel.
- Use minimalist, line-based icons related to personal growth and mindfulness.
- Clean and spacious layout with ample white space to reduce visual clutter and promote focus.
- Gentle animations and transitions to provide a smooth and calming user experience.
